# 메이플 훈트: 미니 액션 RPG

직접 캐릭터를 움직여 몬스터를 처치하고 경험치/골드/아이템을 획득하는 미니 액션 RPG입니다. 드랍 장비를 착용해서 성장할 수 있습니다.

## 실행 방법

```bash
python3 life_simulator.py
```

## 플레이 방법

- 이동: 방향키 또는 WASD
- 공격: 스페이스바
- 몬스터를 처치하면 경험치와 골드, 장비가 드랍됩니다.
- 드랍 장비는 좌측 목록에서 선택 후 `장비 착용` 버튼으로 장착할 수 있습니다.

## 팁

- 장비를 잘 챙기면 공격력과 방어력이 빠르게 올라갑니다.
- 체력이 부족하면 드랍되는 생명의 수정(HP 회복)을 노려보세요.
- 몬스터가 몰려오면 먼저 떨어져서 안전하게 공격하세요.
