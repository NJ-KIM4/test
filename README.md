# 간단한 인생시뮬레이터 키우기 게임

새 창에서 실행되는 간단한 인생 시뮬레이터입니다. 6살부터 25살까지 매년 하나의 행동을 선택하며, 건강/행복/지식/돈 수치를 관리합니다.

## 실행 방법

```bash
python3 life_simulator.py
```

## 플레이 방법

1. 매년 한 가지 행동을 선택합니다.
2. 랜덤 이벤트가 발생해 능력치가 변합니다.
3. 25살이 되거나 건강/행복이 0이 되면 종료됩니다.

## 팁

- 공부와 운동으로 장기적인 성장을 노려보세요.
- 놀기/휴식은 행복과 건강을 회복하는 데 도움이 됩니다.
- 아르바이트는 돈을 늘리지만 체력과 행복에 부담이 있습니다.
